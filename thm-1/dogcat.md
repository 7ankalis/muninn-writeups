* The script checks if the `view` parameter is present in the URL.
* If `view` is present, it checks whether the value of `view` contains the word "dog" or "cat" using the `containsStr` function.
  * If it does, it prints "Here you go!" and includes the file specified by the `view` parameter along with the extension (`$ext`).
  * If it doesn't, it prints "Sorry, only dogs or cats are allowed."
